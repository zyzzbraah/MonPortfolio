Contexte Biologique

Chercher une aiguille dans une botte de foin (ADN)

Un Motif est une courte séquence d'ADN (ex: facteur de transcription) qui se répète et possède une fonction biologique.

  • Problème : Le motif n'est pas identique à chaque fois (mutations). Une simple recherche de texte (CTRL+F) échoue.
  • Données : Nous avons des centaines de séquences bruitées (A, C, G, T) et nous ignorons où est le motif et à quoi il ressemble exactement.

Pour trouver le motif, on apprend à l'ordinateur à fabriquer de fausses séquences.

Program.cs 
// 1. Définition du Priors (La forme du motif est inconnue)
var motifCharsRange = new Range(motifLength);
motifNucleobaseProbs[motifCharsRange] = Variable.Dirichlet(pseudoCounts);

// 2. Définition de la Position (Uniforme)
var sequenceRange = new Range(SequenceCount);
motifPositions[sequenceRange] = Variable.DiscreteUniform(length);

// 3. Construction Générative (Le "Sandwich")
using (Variable.ForEach(sequenceRange))
{
    using (Variable.If(motifPresence[sequenceRange]))
    {
        // On crée le motif à partir des dés pipés (Dirichlet)
        var motif = Variable.StringFromArray(motifChars);
        
        // On l'entoure de bruit (Background)
        sequences[sequenceRange] = backgroundLeft + motif + backgroundRight;
    }
}

Analyse

Plus la séquence est longue, plus la probabilité qu'un motif aléatoire ("faux positif") apparaisse dans le bruit augmente.

Limite observée : Autour de 500 caractères, le signal du motif se dilue trop par rapport au volume de bruit avec seulement 70 séquences.

Expérimentation : Quantité de Données

Combien de séquences minimum faut-il pour apprendre ?

Hauteur = Conférence du modèle (Posterior Mean)

Le modèle converge très vite. Dès 20-30 séquences, le motif émerge clairement du bruit, ce qui prouve la puissance de l'approche bayésienne sur des petits datasets.

Conclusion

Points Forts

  • Code déclaratif et concis.
  • Gère naturellement l'incertitude.
  • Robuste aux petits jeux de données.

Limites

  • Complexité exponentielle avec la longueur.
  • Nécessite de définir les "Priors" correctement.
